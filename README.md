## 新手入门

## 项目运行
1. 将index.php放到服务器目录
1. 数据库目录覆盖mysql的data目录，修改sql/mysql.php和fqxy/sql/mysql.php的数据库配置
1. 配置config/config.php目录下的host域名
1. 通过域名进行访问

## 运行环境
- php 5.6.40
- mysql 5.7.22
